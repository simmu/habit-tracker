# Project rules (read me before doing anything)

Stack: React + Vite (frontend), FastAPI + SQLite (backend).

Always, before saying you're done:
1. Run `npm run lint` and fix every error.
2. Run `npm run typecheck` and fix every error.
3. Run `npm test` and make sure everything passes.
4. Never edit files outside this project folder.

Style: small components, clear names, no dead code.
